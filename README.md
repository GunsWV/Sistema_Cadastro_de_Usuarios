# 📚 Projetos em Python

Este repositório reúne alguns projetos desenvolvidos durante meus estudos em Python, com foco em lógica de programação, manipulação de dados e interação com o usuário via terminal.

---

## 🚀 Projetos

### 🔹 1. Sistema de Cadastro de Usuários
Aplicação em Python com menu interativo que permite:
- Cadastrar usuários
- Remover usuários
- Listar usuários cadastrados

📌 Conceitos utilizados:
- Listas
- Estruturas de repetição (while)
- Condições (if/else)
- Tratamento de erros (try/except)
- Estrutura de decisão (match case)

📁 Arquivo: `Sistema_de_Cadastro.py`

---

### 🔹 2. Gerenciador de Lista de Nomes
Sistema interativo para manipulação de listas, permitindo:
- Adicionar nomes
- Editar nomes
- Remover nomes
- Listar dados
- Limpar a lista

📌 Conceitos utilizados:
- Listas
- enumerate()
- Manipulação de dados
- Validação de entrada

📁 Arquivo: `Gerenciador_de_Lista_de_Nomes.py`

---

### 🔹 3. Conversor de Moeda (USD → BRL)
Aplicação que realiza conversão de valores em dólar para real.

📌 Funcionalidades:
- Conversão de moeda
- Uso de constante para taxa de câmbio
- Validação de entrada
- Execução contínua

📌 Conceitos utilizados:
- Constantes
- Funções
- Entrada e saída de dados
- Tratamento de erros

📁 Arquivo: `Conversor_de_Moeda.py`
